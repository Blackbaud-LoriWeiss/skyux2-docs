By default, SKY UX Builder creates a wildcard route to handle any unknown requested routes - affectionaly known as the "404 page." On the component for this page, we also use an iFrame to display the "Errors SPA", ultimately using the error component with an errorType of notfound.

In order to customize this route, create src/app/not-found.component.ts with the following contents:

import { Component } from '@angular/core'; @Component({ templateUrl: './not-found.component.html' }) export class NotFoundComponent {}

This is my custom 404 page.

Please note, the filename is technically not important to this functionality, but the component name is crucial. It must export NotFoundComponent.
